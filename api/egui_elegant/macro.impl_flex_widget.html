Expand description

Implements [egui_flex::FlexWidget] for a type that implements Elegant.

Due to Rust orphan rules, a blanket impl<W: Elegant> FlexWidget for W is not allowed (neither FlexWidget nor W is local). This macro generates an explicit impl per component.

§Example

ⓘ
impl_flex_widget!(ElegantButton<'a>, 'a);
impl_flex_widget!(Progress); // no lifetime
